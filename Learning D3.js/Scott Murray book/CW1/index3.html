ERROR TYPE: Redundant Data/ Summarization Errors

ALL INDIA and ALL IRAQ are summarization records

The column country in the original provided dataset had entries for ALL INDIA and ALL IRAQ which were basically summarization records

This data was not useful for the visualization, so was removed

The error was detected using a Text Facet on the country column and the entries were deleted using OpenRefine using the remove all matching rows option after applying the filters

ERROR TYPE: Multiple Representations

Example of Multiple Representation on the name column
Example of Multiple Representation on the owner column
More examples of the same error using different keying functions for clustering
Similar errors exist in the source column
UK and GBR: another example of Multiple Representationon the country column

The errors were detected using OpenRefine. Text Facets were applied on each column and clusters were observed

Various Keying Functions were applied to change how aggresively the clustering occured.

fingerprint keying function generally worked well for name column. But metaphone3 worked better for owner and source columns

For the clusters the option: Browse Cluster provided by OpenRefine proved to be very useful as it provided better context and helped ensure that dissimilar values are not clustered together.

OpenRefine provides an option to edit a cell and apply the same change to all identical cells which was used to change the values of UK to GBR

ERROR TYPE: Blank/Missing Values

Example of Blank values in the country_long column
The column fuel4 has all blank values
NaN values present in some generation metrics columns
198 rows in which all numeric generation metrics are blank

The errors were detected using OpenRefine. Text Facets. Blank values were there in country_long column. These were replaced with the correct value by applying two Text Facets - one on the country column and one on the country_long column and then filling the blank value using the correct value observed in the country_long facet OpenRefine provides an option to edit a cell and apply the same change to all identical cells which was used

The column fuel4 was removed as it contained all blank values

NaN values were present in some of the numeric columns. There were not too many of these so these values were replaced by blank values

After replacing the NaN values in the columns, numeric facets were applied on the columns: generation_gwh_2013, generation_gwh_2014, generation_gwh_2015, generation_gwh_2016 and estimated_generation_gwh. It is observed that for 198 rows all these columns are blank. So they will not be contributing anything to the visualization. So they are removed

Visualization 3.2

Scatter Plots to identify how factors like Fighting skills and Energy Projection skills affect popularity


Information about the plot and Conclusions from the above plot:

The Scatter Plots plot the number of appearances on the Y axis and different factors like Strength, Intelligence, Fighting Skills and Energy Projection scores on the X axis. Also it makes use of the size of each circle to indicate a third variable like Durability and Speed.

To prevent overfitting some data points have been manually removed from the plot. This does not affect the overall trends and results of the plot and the statistical measures described below are for the entire dataset

In the above figure, the size of the circles represent Strength score

High Fighting score positively impact Popularity - The R_squared value is 0.04 and the p-value is 0.05, which is significant

High Fighting Skills score does not necessarily mean that the character has high Strength, which seems counterintuitive and is an interesting observation. Some examples are Captain America, Deadpool and Wolverine

It appears as if Energy Projection skill has a negative impact on popularity. This also is slightly strange as Thanos, Thor, Iron Man and Doctor Strange are popular characters

However, the R_squared value is only 0.0023 and the p value is 0.6 which is not significant

Most characters with high Energy Projection skills are also highly strong. This is expected as to emmit high-energy beams, their body must be accustomed for it

Visualization 4

Bubble Chart to visulaize the most powerful heroes and villains and the most powerful male and female characters

Information and Conclusions from the above plot:

In both the plots, Strength is plotted on the X axis and Speed is plotted on the Y axis. In the first plot, the color of the circles represent Character Alignment, while in the second plots the color represents Sex of the characters The size of the circles represent Intelligence score To prevent overlapping and to observe the clusters, a random noise (jitter) has been added to the strength and speed scores. However the tooltip that is generated when a user hovers over a point represents the correct value for the Strength and Speed. Also, users can use the filter to see where their favorite superheroes fall on the plot.

Towards the right-upper part of the plots, we can see a cluster of data points which are large in area. These are the characters who are the most powerful, i.e. have high Strength, Speed and Intelligence

From the first plot we observe that The Watcher, Nova and Rogue are some of the most powerful superheroes

Galactus is one of the powerful characters who is of neutral character alignment

Thanos is the most powerful super villain

From the second plot we observe that, of the most powerful characters, only Rogue is the female

Another interesting observation from this plot is there are very few female characters who are highly intelligent. Thus the comic world may be gender biased and we kind of intuitively know that the most popular intelligent characters like Tony Stark, Dr. Bruce Banners, Hank Pym are all males.
